Question-2: What is the difference between innerHTML, innerText, and textContent ?

textContents is all text contained by an element and all its children that are for formatting purposes only. innerText returns all text contained by an element and all its child elements. innerHtml returns all text, including html tags, that is contained by an element.

Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a technique where a single event listener is attached to a parent element to handle events of its child elements, leveraging event bubbling to efficiently manage events on multiple elements.

Question-4: What is event bubbling in the DOM?

In the DOM, event bubbling is a mechanism where an event triggered on a nested element propagates upwards to its parent and ancestor elements, allowing for centralized event handling.

Question-5: How do you create, add, and remove elements using JavaScript?

Creating Elements

You can create an HTML element using the document.createElement() method.

Adding Elements

appendChild() (Adds as the last child), append() (Can add text and multiple nodes), prepend() (Adds as the first child), insertBefore() (Inserts before a specific element)

Removing Elements

removeChild() (Removes a specific child node) remove() (Removes itself)
